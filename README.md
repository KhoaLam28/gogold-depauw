# GoGold DePauw Racing — Club Website

A responsive, production-deployed website for **GoGold DePauw Racing**, DePauw University's student-run motorsports and engineering club.

**Live Site:** [khoalam28.github.io/gogold-depauw](https://khoalam28.github.io/gogold-depauw)

---

## Overview

Designed and developed from scratch for the GoGold DePauw Racing club. The site serves as the team's public-facing presence — showcasing the team, mission, and contact information to prospective members, sponsors, and the DePauw community.

---

## Features

- **Apple-inspired UI** — clean typography, frosted glass nav, and minimal design system
- **Fully responsive** — mobile-first layout using CSS Grid and Flexbox
- **Custom design system** — CSS variables for consistent gold, black & white branding
- **Smooth scroll navigation** — sticky nav with anchor links
- **Team showcase** — dynamic card grid for all club members and roles
- **Contact form** — integrated with Formspree for backend-free email delivery
- **Deployed via GitHub Pages** — live with CI/CD on every push to `main`

---

## Design Decisions

- **No frameworks** — built with vanilla HTML, CSS, and JavaScript to demonstrate fundamentals
- **CSS custom properties** — centralized brand colors for easy theming and consistency
- **Mobile-first** — responsive breakpoints ensure usability on all screen sizes
- **Performance-first** — no external dependencies or heavy libraries, fast load times

---

## Getting Started

To run locally, simply clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/KhoaLam28/gogold-depauw.git
cd gogold-depauw
open index.html
```

---

## Deployment

The site is deployed via **GitHub Pages** and automatically updates on every push to the `main` branch.
